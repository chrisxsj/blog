## 快捷键

| 快捷键 | 描述                     |
| ------ | ------------------------ |
| ctrl+A | 将光标移到命令行开头     |
| ctrl+E | 将光标移到命令行结尾     |
| ctrl+C | 强制终止当前命令         |
| ctrl+L | 清屏，等于clear          |
| ctrl+U | 删除或剪切光标之前的命令 |
| ctrl+k | 删除或剪切光标之后的内容 |
| ctrl+Y | 粘贴ctrl+k或ctrl+l的内容 |
| ctrl+R | 在历史命令中搜索         |
| ctrl+D | 退出终端                 |
| ctrl+S | 暂停屏幕输出             |
| ctrl+Q | 恢复屏幕输出             |
